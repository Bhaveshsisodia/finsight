## 🚀 Overview

**FinSight India** is a premium, multi-dimensional stock research platform that leverages AI and LangGraph to provide deep insights into Indian stocks. It goes beyond simple price tracking by analyzing fundamentals, technicals, news sentiment, and FII/DII activity in a single, unified dashboard.

Designed for serious investors, FinSight India transforms raw market data into actionable intelligence using a state-of-the-art analytical pipeline.

---

## 🎯 Key Features

### 🔍 Multi-Dimensional Analysis
*   **📊 Fundamental**: Deep dive into profitability, growth, and valuation metrics.
*   **📈 Technical**: Real-time price action analysis, trend detection, and momentum indicators.
*   **📰 News Sentiment**: AI-powered sentiment analysis of the latest market news.
*   **🏛️ FII / DII Activity**: Tracking institutional flows (Net Buying/Selling) via NSDL data.
*   **⚡ Relative Strength**: Benchmarking stock performance against market indices.

### 🤖 Intelligent Insights
*   **Final Score**: A weighted multi-dimensional score (0-100) for at-a-glance evaluation.
*   **Signal Generator**: Clear "BUY", "HOLD", or "SELL" recommendations based on consolidated data.
*   **Confidence levels**: Transparency on how strong the generated signals are.
*   **AI Research Report**: Comprehensive, human-readable summary generated by LLMs.

## 🏗️ Project Architecture

FinSight India uses a modular **LangGraph** workflow to handle complex data dependencies:
1.  **State Management**: Centralized `StockAnalysisState` tracks all analytical dimensions.
2.  **Parallel Nodes**: Independent nodes for Fundamental, Technical, and News analysis run concurrently.
3.  **Synthesis Node**: A final "Probability Node" aggregates all inputs to generate the final signal and report.

---

## ⚙️ Tech Stack

*   **Frontend**: Streamlit (with Custom CSS/JS for premium visuals)
*   **Orchestration**: LangGraph
*   **AI Engine**: LLMs (OpenAI/Groq via LangChain)
*   **Visuals**: Plotly (Radar Charts, Metrics)
*   **Data**: yfinance, NSDL (Scraping), Feedparser (News)

## 🔧 Setup & Installation

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/your-username/finsight-india.git
cd finsight-india
```

### 2️⃣ Fast Launch (Recommended)
We've provided a `start.bat` script that automatically handles environment setup (Python 3.11), dependency installation, and launching the app.

Simply run:
```pwsh
.\start.bat
```

### 3️⃣ Manual Installation
If you prefer manual setup:
1.  **Create a virtual environment**: `python -m venv venv`
2.  **Activate it**: `.\venv\Scripts\activate` (Windows)
3.  **Install requirements**: `pip install -r requirements.txt`
4.  **Launch**: `streamlit run ui/app.py`
